# Audio Files for Sistema Nacho

Place your audio files in the following structure:

## Music (Background loops)
`public/audio/music/`
- `ambient.mp3` - Main background music (dark/epic ambient loop)

## Sound Effects
`public/audio/sfx/`
- `mission_complete.mp3` - When completing a mission
- `level_up.mp3` - When leveling up
- `loot_drop.mp3` - When receiving loot/rewards
- `dungeon_enter.mp3` - When entering a dungeon
- `dungeon_victory.mp3` - When clearing a dungeon
- `shadow_arise.mp3` - When extracting/equipping a shadow
- `click.mp3` - UI button clicks (optional)
- `notification.mp3` - System notifications

## Recommended specs
- Format: MP3 (preferred) or WAV
- Music: 128-192 kbps, loop-friendly
- SFX: Short clips (1-3 seconds)
